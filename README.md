# Simple Calculator

A basic calculator built to perform simple arithmetic operations such as addition, subtraction, multiplication, and division.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- User-friendly input
  
## Technologies Used

- Python (or replace with your programming language)

## How to Run

1. Make sure Python is installed.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the program:

```bash
python calculator.py
```

## Example

```
Enter first number: 15
Enter operator (+, -, *, /): *
Enter second number: 8

Result: 120
```
